# Сайт-визитка амбассадора Магнит — v3

Статический сайт без сборки. Можно просто открыть `index.html` в браузере или загрузить файлы на GitHub Pages.

## Что добавлено в этой версии

- Новый раздел **«Моё резюме»**.
- Опыт работы в **АО ТАНДЕР**.
- Образование: **СПО — Юриспруденция**.
- Красивый блок навыков с анимированным графиком по **Excel / таблицам**.
- Добавлены навыки: **Web-дизайн**, **кодинг**, **вайбкодинг**.
- Анимированные skill bars.
- Отдельная вкладка **«Резюме»** в боковой колонке.
- Сохранены:
  - светлая / тёмная тема;
  - адаптив под телефон;
  - модальные окна у новостей;
  - галерея фотографий;
  - анимации и минималистичный стиль.

## Где менять данные

- Главное фото: в `index.html` найди первое `<img src="https://i.imgur.com/...">`.
- Фото в галерее: секция `#gallery`.
- Резюме: секция `#resume`.
- Навыки: в `#resume` поменяй `data-progress` и проценты в строках.

## GitHub Pages

1. Загрузи в репозиторий:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
2. Открой **Settings → Pages**.
3. Выбери **Deploy from a branch**.
4. Укажи **main** и **/root**.
5. Сохрани.
